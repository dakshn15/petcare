# Petcare | Professional Pet Care & Grooming Services

A modern, responsive, and interactive Single Page Application (SPA) for **Petcare**, rebuilt from scratch using **React 19**, **Tailwind CSS**, and **Vite**, with full Progressive Web App (**PWA**) support. 

This project was successfully migrated from a static HTML/CSS/JS template. All instances of "WorkDo" branding, copyrights, and external links have been completely removed and replaced with clean, customizable components.

---

## 🚀 Key Features

*   **Modern Single-Page Navigation**: Fast, fluid routing powered by `react-router-dom` with automatic scroll restoration.
*   **Tailwind CSS styling**: Beautiful styling matching the original design language exactly, utilizing unified color variables and responsive utility layouts.
*   **Progressive Web App (PWA)**:
    *   **Install Prompt**: Floating desktop/mobile installation banner prompt (`InstallButton.jsx`).
    *   **Offline Support**: Fully functional service worker and resource caching generated by `vite-plugin-pwa`.
    *   **In-App Updates**: A toast notification prompting users to refresh when new application versions are published (`ReloadPrompt.jsx`).
*   **Interactive UI Components**:
    *   **Animated Counter**: Count-up animation for company statistics (`Counter.jsx`).
    *   **FAQ Accordions**: Smooth expand/collapse FAQ categorizations (`Accordion.jsx`).
    *   **Add-On Price Calculator**: Dynamic price calculating triggers on grooming services add-ons.
    *   **Customer Reviews Feed**: Interactive review submission form with automatic rating and success confetti.
    *   **Swiper Carousels**: Multi-card responsive sliders for team experts, testimonial cards, pet profiles, and hero banners.
*   **Client-Side Validation**: Complete, interactive form validations and custom success notifications for Service Booking, Adoption, and Contact messages.

## 🛠️ Technology Stack

*   **Framework**: [React 19](https://react.dev/)

## ⚙️ Development & Scripts

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
Clone the repository, navigate to the folder, and install the dependencies:
```bash
npm install
```

### Development Server
Run the local dev server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

### Build and Compilation
Build the production-ready bundle. This runs Vite's build, compiles tailwind styles, processes assets, and generates the service worker (`dist/sw.js`):
```bash
npm run build
```

### Preview Production Build
Locally preview the built application assets:
```bash
npm run preview
```

## 📋 Custom PWA Configuration

To handle larger images (such as `cat.png` at 2.13 MB) in precaching without triggering build warnings, the Workbox configuration in `vite.config.js` is customized:
```javascript
workbox: {
  maximumFileSizeToCacheInBytes: 5 * 1024 * 1024 // 5 MiB precache threshold
}
```
